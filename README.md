# 🤍 CarLi-Booth

> *Capturing little moments, one click at a time.*

A full-stack web application created to celebrate my cousins' wedding by turning a simple web browser into a DIY digital photobooth. Guests can pose, capture memories, choose beautiful wedding frames, and take home a keepsake—all from one website.

This project is also part of my journey in learning modern full-stack web development using **React**, **Vite**, and **Node.js**.

---

## ✨ About the Project

Every wedding is filled with moments worth remembering.

This project aims to create a simple, elegant, and interactive photobooth experience that allows guests to capture memories effortlessly while adding a personal touch through custom-designed wedding frames.

Beyond the celebration, this project serves as a hands-on opportunity to explore full-stack development, user experience design, and camera integration on the web.

---

## 🌸 Features

Current and planned features include:

* 📷 Live camera preview
* ⏳ Countdown before capture
* 📸 Take high-quality photos
* 🖼️ Custom wedding photo frames
* 💾 Download captured photos
* 📱 Responsive design for desktop and mobile
* 🗂️ Online gallery

## 🌷 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/JHSalgado4/CarLi-Booth.git
```

### 2. Open the project

```bash
cd CarLi-Booth
```

### 3. Install the dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd ../server
npm install
```

### 4. Start the development servers

Frontend

```bash
npm run dev
```

Backend

```bash
npm run dev
```
